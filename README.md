## 👋 Introduction

This project is a very basic Windows desktop utility built with WinForms and VB.NET. Originally developed back in 2012, it was the literal first program I ever wrote in Visual Studio. Its only function is to track and display the real-time X and Y screen coordinates of your mouse cursor.

Please note that this repository is kept mostly as a personal memento and for historical reference of my early coding steps, so probably it will not receive updates or code refactoring.

## 💡 Motivation

The motivation behind this project was simply to learn how to program. Back in 2012, I was taking my very first steps in development and needed a real, hands-on project to understand how the .NET framework, variables, events, and a graphical interface worked. This tiny utility was the result of that initial learning process.

###### Every programmer has a starting point; what was the very first piece of code you compiled when you started learning?.
